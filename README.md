# PODPOD 经济模型测算

调整参数，实时查看三方收益与投入回收周期。

## 在线访问

部署到 GitHub Pages 后，链接格式为：
```
https://你的用户名.github.io/仓库名
```

## 部署步骤

### 第一次部署

1. 在 GitHub 新建一个仓库（例如 `podpod-model`）
2. 把这两个文件上传进去：`index.html` 和 `README.md`
3. 进入仓库 → **Settings** → **Pages**
4. Source 选择 **Deploy from a branch**，Branch 选 **main**，目录选 **/ (root)**
5. 点 Save，等 1–2 分钟，页面就上线了

### 后续更新

直接在 GitHub 网页上编辑 `index.html`，保存后 1–2 分钟自动同步。

---

## 功能说明

- 三场景快速切换（保守 / 基准 / 乐观）
- 流量、分成、AI 成本、初次投入等所有参数可拖动调整
- 初次投入可指定由景区 / 运营方 / PODPOD 任一方承担
- 日 / 月 / 年三个维度切换查看收益
- 支持保存当前方案，历史记录最多保留 20 条
- 历史记录支持一键恢复参数，支持单条删除
